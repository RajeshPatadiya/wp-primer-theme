Function primer_excerpt_length

Set the post excerpt length to 20 words.

To override this in a child theme, remove the filter and add your own function tied to the excerpt_length filter hook:

remove_filter( 'excerpt_length', 'primer_excerpt_length' );
add_filter( 'excerpt_length', function() { return 30; } );
